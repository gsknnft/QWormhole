# SCP Negentropy Vector

## Overview
Negentropy Vector (NV) is a compressed representation of agent entropy, combining:
- Shannon entropy
- Kolmogorov complexity
- Spectral flow entropy
- Social/cosmic jitter

## NV Structure
```json
{
  "shannon": 0.98,
  "kolmogorov": 0.87,
  "spectral": 0.92,
  "jitter": 0.05
}
```

## Usage
- Peer stability detection
- Coherence and drift analysis
- Hijack/impersonation/decay detection

## Example
```json
{
  "shannon": 0.99,
  "kolmogorov": 0.88,
  "spectral": 0.93,
  "jitter": 0.04
}
```
